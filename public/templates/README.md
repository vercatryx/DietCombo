# Templates Directory

This directory contains PDF templates used by the application.

## Template Files

The following template files are used:
- `XHRTEMP.pdf` - XHR template (renamed from "XHR TEMPLATE.pdf")
- `CertTEMP.pdf` - Certificate template (renamed from "CERT TEMPLATE.pdf")

These files should be accessible at:
- `/templates/XHRTEMP.pdf`
- `/templates/CertTEMP.pdf`
